# Amazon Prime Sorter - Icon

## How to Create the Icon

Since we're keeping this extension dependency-free, here are two options:

### Option 1: Use Online Tool (Recommended)

1. Go to https://favicon.io/favicon-generator/
2. Create a 128x128 PNG with:
   - Background: Dark blue gradient (#232f3e to #37475a)
   - Text: "A" in Amazon orange (#FF9900)
   - Rounded corners (radius: 24px)
3. Download and save as `icon128.png` in the `icons/` folder

### Option 2: Use the HTML Preview

1. Open `icon-preview.html` in your browser
2. Take a screenshot
3. Crop to 128x128
4. Save as PNG

### Option 3: Use This Base64 (Quick Start)

A simple placeholder icon has been created below.

## Icon Requirements

- **Size**: 128x128 pixels (PNG format)
- **Design**: Letter "A" in Amazon orange (#FF9900) on dark blue background
- **Style**: Modern, rounded corners, professional look
